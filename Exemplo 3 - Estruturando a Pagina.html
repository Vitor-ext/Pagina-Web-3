Computação em Nuvem AWS

Curso de AWS

O Curso de Aperfeiçoamento Profissional de Implantação de Serviços
em Nuvem - AWS Cloud Practitioner Foundational tem por objetivo o
desenvolvimento de competências relativas à implantação e gerenciamento
de serviços em nuvem, por meio dos conteúdos formativos da AWS Foundation,
possibilitando que o profissional pleiteie a certificação AWS Cloud Practitioner Foundational.

...

Programação do curso:

  • Configurar redes de microcomputadores;
  • Identificar características de serviços em nuvem;
  • Configurar máquina virtual (IaaS) - infraestrutura como serviço;
  • Configurar web server (PaaS) - plataforma como serviço;
  • Configurar armazenamento em nuvem (SaaS) - software como serviço;
  • Configurar soluções de segurança de serviços em nuvem;
  • Integrar serviços de plataforma em nuvem;
  • Definir modelos de contratação de serviços em nuvem;
  • Monitorar serviços em nuvem;
  • Redefinir configurações de serviços em nuvem;
  • Demonstração raciocínio lógico;
  • Demonstrar atenção aos detalhes.


    • Computação em Nuvem Google

    Curso de iCloud Google

    O curso de Implantação de Serviços em Nuvem - Google Cloud Foundations
    tem por objetivo preparar o estudante para certificações Cloud Digital Leader,
    permitindo articular os recursos dos principais produtos e serviços do Google Cloud,
    desenvolvendo capacidades que possibilitem ao concluinte implementar serviços de
    armazenamento e análise de dados, utilizando as principais tecnologias da área

    ...

    Programação do curso:

  • Reconhecer as características de arquitetura de nuvem e arquitetura tradicional;
  • Controlar acesso por meio do Cloud Identity and Access Management (IAM);
  • Acessar linha de comando via interface do GCloud utilizando Cloud Shell e ferramentas de automação e gerenciamento;
  • Reconhecer aplicações do Aplication Programming Interface (API) da Google Cloud;
  • Criar e executar máquinas virtuais dentro da infraestrutura do Google - Compute Engine;
  • Configurar e disponibilizar as principais formas de armazenamento de dados em nuvem;
  • Realizar segurança e distribuição no acesso a Cloud;
  • Configurar o Software Development Kit (SDK) do Cloud;
  • Executar código de linguagem de programação PaaS com AppEngine;
  • Configurar e executar ambientes de BigData;
  • Configurar e executar ambiente de Machine Learning (ML);
  • Gerenciar faturamento.
